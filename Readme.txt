Language Used : C++ 


How to Run : 

1) Open Visual Studio 2017 Command Prompt. You can't use Visual C++ in a plain command prompt window.If you search 
   for 'developer command prompt' under the start menu (Provided, you have installed Visual Studio 2017 or Visual CPP)
   , you will be able to find the required command prompt.

2) Navigate to the folder where the sourcefile has been placed.
3) Enter the following command to compile the source file.

   cl /EHsc MiniMaxOpening.cpp

4) Run the program using the command given below.

   MiniMaxOpening board1.txt board2.txt <depth>
